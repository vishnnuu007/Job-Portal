# Job Portal System

## Overview

Job Portal System is a web application developed using ASP.NET Core Web API and SQL Server. The system allows job seekers to search and apply for jobs while employers can post and manage job listings.

## Features

* User Registration and Login
* JWT Authentication
* Job Posting Management
* Job Search Functionality
* Apply for Jobs
* Employer Job Management
* RESTful API Architecture
* SQL Server Database Integration

## Technologies Used

* ASP.NET Core Web API
* C#
* Entity Framework Core
* SQL Server
* JWT Authentication
* Repository Pattern
* Service Pattern

## Project Structure

* Controllers
* Models
* Services
* Repositories
* Data Access Layer
* Authentication Module

## Setup Instructions

1. Clone the repository
2. Open the project in Visual Studio
3. Configure the database connection string
4. Run Entity Framework migrations
5. Build and run the project
